**Training Parameters:**
- `--algorithm`: `rf`, `svm`, `xgboost`, `mlp`
- `--cv-folds`: Cross-validation folds (default: 5)
- `--optimize`: Enable hyperparameter tuning

                            --batch data/samples/ \
                            --output results.csv
